String comparison operations:=
	str1.equals(str2)- Case sensitive
	str1.equalsIgnoreCase(str2)- Not case sensitive
	str1.compareTo(str2)- returns 0 if both strings are same
				    returns + number if str1 is bigger than str2
                            returns - number if str2 is bigger than str1
	str1.compareToIgnoreCase(str2)- Not case sensitive.
	str1.matches(regex)- matches the string to regular expressions , returns true if matches otherwise false.
